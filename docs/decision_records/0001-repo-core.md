# Decision 0001: crear repositorio IA_NEST Core

Fecha: 2026-07-10

## Decision

Crear un repositorio nuevo para definir y estabilizar el core de IA_NEST.

## Motivo

El repositorio IA_NEST actual contiene demasiado alcance acumulado. El nuevo
repositorio permite separar el nucleo conceptual de las integraciones y modulos
perifericos.

## Consecuencia

El codigo existente se tratara como cantera, no como base a migrar de forma
automatica.
