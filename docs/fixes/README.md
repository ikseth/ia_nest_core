# Correcciones y mejoras pequenas

Este directorio conserva el contexto de cambios que necesitan mas detalle que
una entrada de `CHANGELOG.md`, pero no una decision estructural en forma de ADR.

Las fichas se agrupan por linea `MAJOR.MINOR` y se numeran dentro de ella:

    docs/fixes/v0.1/0001-descripcion-breve.md

Cada ficha indica estado, tipo, impacto de version, version objetivo, problema,
cambio, criterios de aceptacion, archivos previstos y resultado. La regla
completa y el criterio para distinguir una ficha de un ADR viven en
`docs/VERSIONADO.md`.
