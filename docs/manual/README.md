# Manual de usuario de IA_NEST Core

Manuales cortos y modulares: una tarea por pagina. Para no ahogarse en
documentacion, la regla es: si algo crece, se parte en otra pagina; no se
engorda una existente.

## Paginas

- [instalacion.md](instalacion.md) - instalar desde cero.
- [configuracion.md](configuracion.md) - modelos, dominios y perfiles.
- [cli.md](cli.md) - comandos de la CLI (referencia rapida).
- [interfaces.md](interfaces.md) - arrancar y usar REST y MCP.

Para desplegar en un host con GPU, ver tambien [../DESPLIEGUE.md](../DESPLIEGUE.md).

## Convencion

- Cada pagina es corta y de una sola tarea (referencia, no tratado).
- Capacidad nueva = pagina o seccion nueva pequena, NO refactor de un
  documento grande.
- Los comandos son reales y verificados.
